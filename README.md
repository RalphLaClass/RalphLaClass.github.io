## Présentation

Portfolio vitrine développé from scratch — sans framework, sans template.  
Design inspiré de l'environnement terminal Kali Linux : fond sombre, vert terminal, typographie monospace.

Conçu pour présenter mon profil à des recruteurs en cybersécurité et maximiser mes chances de trouver une alternance SOC / Blue Team dès septembre 2025.

---

## Structure

```
portfolio/
├── index.html      ← Structure complète du site (5 sections)
├── style.css       ← Design responsive, variables CSS, animations
└── script.js       ← Cursor personnalisé, terminal animé, scroll reveal
```

## Sections

- **Hero** — Nom, titre, terminal Kali animé, stats TryHackMe / Root-Me
- **À propos** — Parcours, disponibilité, plateformes CTF
- **Compétences** — Sécurité offensive, SOC & Défense, Développement, Systèmes & Réseaux
- **Projets** — 8 projets avec liens GitHub
- **Contact** — Email, GitHub, LinkedIn, TryHackMe

---

## Fonctionnalités

- Terminal Kali Linux animé (effet typewriter avec prompt authentique)
- Cursor personnalisé avec trail
- Scroll reveal sur toutes les sections
- Navigation fixe avec blur
- 100% responsive mobile

---

## Lancer en local

Aucune dépendance, aucun build. Ouvre simplement `index.html` dans un navigateur :

```bash
git clone https://github.com/RalphLaClass/RalphLaClass.github.io
cd RalphLaClass.github.io
open index.html   # ou double-clic sur le fichier
```
